**Priority:** Високий <br>

| Step | Test Step               | Test Data       |
| ---- | ----------------------- | --------------- |

## Defect ID 2

**Summary:** Не надходить лист для скидання пароля <br>
**Priority:** Високий <br>

| Step | Test Step                              | Test Data       |
| ---- | -------------------------------------- | --------------- |
| 1    | Відкрити форму "Забули пароль"         |                 |
| 2    | Ввести email                           | email@email.com |
| 3    | Ввести капчу                           |                 |
| 4    | Натиснути "Скинути пароль"             |                 |
| 5    | Перевірити вхідні на поштовій скриньці |                 |

**Expected result:** <br>
На email надходить лист із посиланням для скидання пароля.

## Defect ID 3

**Summary:** Час відповіді при вході перевищує поріг <br>
**Priority:** Високий <br>

| Step | Test Step                                                               | Test Data     |
| ---- | ----------------------------------------------------------------------- | ------------- |
| 1    | 100 тест-користувачів одночасно входять/виходять з валідними даними     | JMeter-скрипт |
| 2    | Запуск протягом якогось часу                                            | 5 хв          |
| 3    | Скрипт збирає кількість входів та виходів за певний час та показує звіт |               |

**Expected result:** <br>
Cередній час входу < 2 секунди, виходу < 1 секунди

**Actual result:** <br>
Середній входу час ~5 секунд, є випадки з затримкою понад 10 секунд.

| Step | Test Step                            | Test Data       |
| ---- | ------------------------------------ | --------------- |

| 2    | Ввести валідний email                | email@email.com |
